What is nnForge?

nnForge is a library for training convolutional and fully-connected neural networks. It includes CPU and GPU (CUDA) backends.

The library implements Stochastic Diagonal Levenberg-Marquardt training algorithm, well described in:

  • Y. LeCun, L. Bottou, Y. Bengio and P. Haffner: Gradient-Based Learning Applied to Document Recognition, Proceedings of the IEEE, 86(11):2278-2324, November 1998
  • Y. LeCun, L. Bottou, G. Orr and K. Muller: Efficient BackProp, in Orr, G. and Muller K. (Eds), Neural Networks: Tricks of the trade, Springer, 1998

You will find these papers at Yann LeCun site.

It also has "dropout" regularization implemented:

  • Improving neural networks by preventing co-adaptation of feature detectors, Geoffery Hinton, Nitish Srivastava, Alex Krizhevsky, Ilya Sutskever, and Ruslan Salakhutdinov, arXiv, 2012

The package contains not only nnForge library but also examples - applications using the library.

Prerequisites

The library depends on Boost and OpenCV.

If you want to use CUDA backend you will also need CUDA Toolkit installed.
