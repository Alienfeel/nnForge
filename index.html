nnForge

Convolutional and fully-connected neural networks C++ library

What is nnForge?

nnForge is a library for training convolutional and fully-connected neural networks. It includes CPU and GPU (CUDA) backends.

Layers

The library has a number of layers defined:

  • Convolutional (1D, 2D, 3D, and 4D)
  • Fully connected
  • Local contrast subtractive (CPU backend - no limitations, GPU backend - 2D only)
  • RGB->YUV conversion (CPU backend: forward propagation only, GPU backend - forward + backward prop)
  • Max spatial subsampling (CPU backend - no limitations, GPU backend - 2D and 3D only)
  • Average spatial subsampling (CPU backend - no limitations, GPU backend - 2D only)
  • Maxout - arXiv:1302.4389
  • Rectification - |x|
  • Hyperbolic tangent - 1.7159*tanh(0.666666*x)
  • Rectified linear unit - max(0,x)
  • Soft rectified linear unit - log(1+exp(x))
  • Softmax

Training

The library implements the following training algorithms:

  • Stochastic Gradient Descent
  • Stochastic Diagonal Levenberg-Marquardt, well described in Y. LeCun, L. Bottou, Y. Bengio and P. Haffner: Gradient-Based Learning Applied to Document Recognition, Proceedings of the IEEE, 86(11):2278-2324, November 1998 and Y. LeCun, L. Bottou, G. Orr and K. Muller: Efficient BackProp, in Orr, G. and Muller K. (Eds), Neural Networks: Tricks of the trade, Springer, 1998. You will find these papers at Yann LeCun site.

Regularization

The library also has "dropout" and "Upper bound on L2 norm of the weight vector" regularizations implemented:

  • Improving neural networks by preventing co-adaptation of feature detectors, Geoffery Hinton, Nitish Srivastava, Alex Krizhevsky, Ilya Sutskever, and Ruslan Salakhutdinov, arXiv:1207.0580

Error functions

Error functions available:

  • MSE - Mean Squared Error
  • SHL - Squared Hinge Loss
  • CE - Cross Entropy

Download

The package contains nnForge library as well as examples - applications using the library.

Prerequisites

The library depends on Boost and OpenCV.

If you want to use CUDA backend you will also need CUDA Toolkit installed.
