nnForge

nnForge is a library for training convolutional and fully-connected neural networks. It includes CPU and GPU (CUDA) backends. nnForge is an open-source software distributed under the Apache License v2.0.

The library depends on Boost and OpenCV.

If you want to use CUDA backend you will also need CUDA Toolkit installed.

The package contains not only nnForge library but also examples - applications using the library.
